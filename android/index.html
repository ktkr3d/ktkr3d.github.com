Android

2013年08月04日

Android のバージョンとAPI レベル

http://source.android.com/source/build-numbers.html

Android SDK の設定

  • ダウンロードと展開

    http://developer.android.com/sdk/index.html –> /tools/android-sdk-linux

  • 環境変数PATH の追加

      $ export PATH=/tools/android-sdk-linux/tools:$PATH
  $ export PATH=/tools/android-sdk-linux/platform-tools:$PATH

  • アップデート

      $ android update sdk
  $ android list target

Android NDK の設定

  • ダウンロードと展開

    http://developer.android.com/sdk/ndk/index.html –> /tools/android-ndk-r9

  • 環境変数PATH の追加

      $ export PATH=/tools/android-ndk-r9:$PATH

build.xml, local.properties, project.properties の生成

android update project —path . —target android-10

デバッグ

  • ログの取得開始

    $ adb logcat -s <Tag名>:* Android 端末でアプリケーションを実行する

  • ログの取得終了

    Ctrl-C

Tips

  • フルスクリーン表示にするには

    AndroidManifest.xml でactivity タグのandroid:theme 属性を設定

実機のスクリーンキャプチャ

Wi-Fi 接続してから、Android SDK をインストールしたPC から

$ export ADBHOST=192.168.0.4
$ adb devices
* daemon not running. starting it now on port 5037 *
* daemon started successfully *
List of devices attached
emulator-5554 device
$ ddms

でddms を起動して、「Device」-「Screen Capture…」メニューから取得する。
