Compiling Irrlicht Ogl-es Branch for Android

2013年08月04日

  • Compiling Irrlicht ogl-es branch for Android on Ubuntu 13.10 x86_64.
  • Native C/C++ Android Application without Java.
  • Fullscreen
  • Input Event Receiver

Environment

  • Android SDK: 22.0.5
  • Android NDK: r9
  • Android Device: Android 4.1.2 (API level 16)

Download Irrlicht ogl-es branch

http://sourceforge.net/p/irrlicht/code/4568/tree/branches/ogl-es/

Download Snapshot
extract

Compile Irrlicht Library

$ cd source/irrlicht/Andorid
$ ndk-build -j4

–> lib/Android/libirrlicht.a

Build example

$ cd examples/17.HellowWorld_Mobile
$ android update project --path . --target android-16
$ ndk-build -j4
$ ant debug

–> examples/17.HellowWorld_Mobile/HelloWorldMobile-debug.apk
