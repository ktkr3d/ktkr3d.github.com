GNOME Shell 3.4 + Autohide Top Panel Extension

2012年04月19日

ここ数日のアップデートでUbuntu 12.04 のGNOME Shell が安定しだしたので、個人的に最も重要なAutohide Top Panel extension を動くようにしてみた。手順は3.2 用のextension を導入して、対応バージョン情報を変更するだけ。

[plain title=“/usr/share/gnome-shell/extensions/autohidetopbar2@werewolves.us/metadata.json” highlight=“2”] { “shell-version”: [ “3.4”, “3.2.0”, “3.2”, “3.2.1” ], “uuid”: “autohidetopbar2@werewolves.us”, “name”: “Auto Hide Top Panel”, “description”: “Auto hide the top panel”, “url”: “http://fpmurphy.com/gnome-shell-extensions”, “original-authors”: [ “Finnbarr P. Murphy”, “Kevin R. Kane” ], “locale”: “locale” } [/plain]

上記の修正をした後、セッションを再起動して、拡張を有効にする。パネルをダブルクリックすると、自動的に隠すか、常時表示するかを切り替えられる。後はConky が安定したらUbuntu 12.04 + GNOME Shell に移行してもよさそう。
