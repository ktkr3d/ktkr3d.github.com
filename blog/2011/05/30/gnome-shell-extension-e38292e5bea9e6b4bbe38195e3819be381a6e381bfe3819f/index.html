GNOME Shell Extension を復活させてみた

2011年05月30日

GNOME Shell を3.0.2 にしたらdock が消えたので復活させてみた。

  • Ubuntu 11.04 の場合 /usr/local/share/gnome-shell/extensions/dock@gnome-shell-extensions.gnome.org/metadata.json

  • Fedora 15, openSUSE 11.4 の場合 /usr/share/gnome-shell/extensions/dock@gnome-shell-extensions.gnome.org/metadata.json

[plain highlight=“6”] { “uuid”: “dock@gnome-shell-extensions.gnome.org”, “name”: “Dock”, “description”: “A dock for the GNOME Shell — displays favorite and running applications”, “original-author”: “tclaesson@gmail.com”, “shell-version”: [ “3.0.1” ], “localedir”: “/usr/local/share/locale”, “url”: “http://git.gnome.org/gnome-shell-extensions” } [/plain] を以下のように修正した。 [plain firstline=“6” highlight=“6”] “shell-version”: [ “3.0” ], [/plain] または、 [plain firstline=“6” highlight=“6”] “shell-version”: [ “3.0.1”, “3.0.2” ], [/plain]
