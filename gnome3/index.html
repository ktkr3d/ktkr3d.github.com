GNOME3

2011-05-15

http://www.gnome3.org/

ここではUbuntu ベースのディストリビューションのデスクトップ環境をGNOME Shell (GNOME 3.6)にしていきます。OS とデスクトップ環境の導入、GNOME Tweak Tool の設定、GNOME Shell Extension の導入、GDM の設定、Nautilus 3.6 の設定、GNOME Shell の操作の順番です。

OS とデスクトップ環境の導入

  1. Ubuntu ベースのOS をインストールします。

    • Ubuntu の場合 http://cdimage.ubuntu.com/releases/
* Ubuntu GNOME Remix の場合

https://wiki.ubuntu.com/UbuntuGNOME/

  1. リポジトリにppa:gnome3-team/gnome3 を追加します。

GTK+ テーマの設定

http://gnome-look.org/content/show.php/?content=157986

gnome-tweak-tools を起動して、「GTK+ テーマ」と「Current theme」で「Uncomplicated」を選択します。

GNOME Tweak Tool の設定

https://live.gnome.org/GnomeTweakTool

  • テーマやフォントを変更します。

  • ウィンドウの最小化ボタン、最大化ボタンを有効にします。

  • 時計に日付を表示します。

  • GNOME Shell Extension を有効にします。 有効にしたい拡張が一覧に表示されない場合はコマンドラインで有効にする。

GNOME Shell Extension の導入

https://extensions.gnome.org/
https://live.gnome.org/GnomeShell/Extensions

  • Autohide Top Panel https://launchpad.net/~webupd8team/+archive/gnome3/ 上部のパネルを自動的に隠してくれます。パネルをダブルクリックすることで表示/自動非表示を切り替えられます。

  • Drop Down Terminal https://extensions.gnome.org/extension/442/drop-down-terminal/ [F12]キーを押下することでターミナルを表示/非表示します。

GDM の設定

  • GDM とLightDM の切り替え
  • GDM の外観(背景、テーマ)の変更
  • GDM の自動ログイン設定

Nautilus 3.6 の設定

日付の書式をYYYY-MM-DD hh:mm:ss に変更します。

GNOME Shell の操作

https://live.gnome.org/GnomeShell/CheatSheet

  • Overview 画面に切り替える 画面左上隅にマウスを移動、システム(Windows)キーまたは[Alt]+[F1]

  • スクリーンキャストの撮影 Ctrl+Alt+Shift+R で開始&終了。WEBM 形式で出力される。

  • ワークスペースの移動 [Ctrl]+[Alt]+[Up]、[Ctrl]+[Alt]+[Down]

  • スクリーンのロック [Ctrl]+[Alt]+[L]

  • GNOME Shell を再起動 [Alt]+[F2] を押して、r を入力して、[Enter] を押す。
